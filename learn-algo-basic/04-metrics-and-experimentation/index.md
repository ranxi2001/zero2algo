# 指标、实验与收益归因

算法项目最常见的失分点不是模型细节，而是讲不清“为什么这个优化真的有效”。

这篇会把离线评估、在线实验和收益归因放到同一个框架里。

## 这篇会覆盖什么

- offline 指标和 online 指标分别回答什么问题
- ablation、A/B 实验和收益归因怎么串起来
- 为什么“指标涨了”不等于“业务真的赚了”
