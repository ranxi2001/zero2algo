# 广告投放链路：定向 / 竞价 / 预算 / 结算

广告算法和推荐排序相似，但多了广告主、预算、出价、拍卖和结算。它不是简单“把广告排在前面”，而是在用户体验、广告主效果和平台收入之间做平衡。

## 广告系统的核心角色

广告系统至少有三方：

- 用户：希望看到相关、不打扰的广告。
- 广告主：希望花钱带来转化和 ROI。
- 平台：希望提升收入，同时维护生态和体验。

算法要同时考虑三方目标。

## 基础链路

典型广告投放链路：

```text
广告主建计划 -> 定向 -> 召回 -> 预估 -> 竞价排序 -> 预算控制 -> 展示点击 -> 转化归因 -> 结算
```

每一环都会影响最终效果。

## 定向

定向决定广告能参与哪些流量竞争。

常见定向：

- 人群定向。
- 地域定向。
- 兴趣定向。
- 关键词定向。
- 类目定向。
- 重定向。
- lookalike 扩量。

定向太窄，消耗不足；定向太宽，转化差。算法要帮助广告主找到有效人群。

## 预估

广告排序通常依赖 CTR、CVR、转化价值等预估。

例如：

```text
eCPM = bid * pCTR * pCVR * value
```

不同计费方式公式不同，但核心都是估计一次曝光的期望价值。

预估偏差会直接影响排序和收费。如果 pCTR 高估，广告可能过度展示；如果 pCVR 低估，优质广告可能拿不到流量。

## 竞价和排序

广告排序不是只按出价。高出价低质量广告不一定赢，因为用户体验和转化概率也重要。

常见排序因子：

- 出价。
- CTR / CVR。
- 广告质量。
- 用户体验。
- 预算状态。
- 频控和策略规则。

面试里要强调：广告系统排序是机制问题，不只是模型问题。

## 预算控制

广告主预算有限，不能一开始就花完，也不能花不出去。

预算控制要解决：

- 一天预算如何平滑消耗。
- 高价值流量是否优先拿。
- 消耗过快如何降速。
- 消耗不足如何扩量。
- 预算和 ROI 如何平衡。

这就是 budget pacing。

## 结算和归因

广告计费方式包括 CPM、CPC、CPA、oCPC、oCPM 等。不同计费方式下，平台和广告主承担的风险不同。

归因回答“哪个广告带来了转化”。常见问题包括转化延迟、多触点归因、跨设备归因、作弊流量和归因窗口。

## 面试题：广告和推荐有什么区别

理想回答：

> 推荐主要优化用户体验和内容消费，广告还要考虑广告主预算、出价、结算和 ROI。广告排序不仅是相关性排序，还涉及拍卖机制和预算控制。广告系统要同时平衡用户、广告主和平台三方目标。

## 面试题：广告链路里算法能做什么

可以回答：

- 定向和扩量。
- CTR/CVR 预估。
- 出价优化。
- 预算 pacing。
- 广告质量评估。
- 反作弊和风控。
- 转化归因。

## 总结

广告投放链路的关键词是：定向、预估、竞价、预算、结算、归因。准备广告算法面试时，不要只讲 CTR 模型，要把商业机制和预算约束讲清楚。
