# 什么是算法岗

这一篇先回答一个最基础但最容易被讲虚的问题：算法岗到底在交付什么。

它不是“会调模型”这么简单，而是围绕业务目标、数据、指标和系统约束持续做优化。

## 这篇会覆盖什么

- 推荐、广告、风控、优化、NLP / AIGC 岗位的共性
- 算法岗常见交付物：模型、策略、实验、监控、复盘
- 为什么面试官更关心你怎么定义问题，而不只是你用了什么模型

## 当前提纲

- 岗位边界与常见误解
- 业务目标、目标函数和收益归因
- 从“模型实验”到“可上线方案”的差别
