# 项目介绍怎么讲

项目介绍是算法面试里最重要的入口。

讲得散，后面的技术追问会失去主线；讲得清楚，面试官会自然顺着你的设计继续问。

## 这篇会覆盖什么

- 项目背景、问题定义、方案、指标、收益、复盘怎么组织
- 如何避免只讲模型细节，不讲业务价值
- 面试官通常会沿着哪些方向继续追问

## 当前提纲

- 项目叙事结构
- 高频追问入口
- 常见失分点
