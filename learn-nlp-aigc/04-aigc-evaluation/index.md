# AIGC 评测与效果对齐

这篇会把质量、稳定性、可控性和成本放进一个更完整的评测框架里。

- 人工评测、自动评测、pairwise 评测的区别
- 幻觉、格式稳定性、风格一致性、拒答策略如何评估
- 为什么效果优化常常和成本优化绑在一起
