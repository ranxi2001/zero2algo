# 规则系统与模型系统协同

规则、模型、策略阈值和人工审核通常要一起工作，才能兼顾稳定性和迭代效率。

- 规则适合解决什么，模型适合解决什么
- 规则和模型如何做分层、兜底和冲突处理
- 阈值、灰名单、黑名单和 case 回流的关系
