# 工业界怎么定义状态、动作、奖励

强化学习真正难的地方通常不是算法公式，而是建模。

状态、动作、奖励一旦定义错了，后面的训练和部署都会偏。

## 这篇会覆盖什么

- 推荐、广告、策略控制里常见的状态设计
- 动作空间过大时常见的降维思路
- 奖励延迟、稀疏、作弊和短视问题
