# 特征工程与样本构造

很多排序问题最后不是输在模型，而是输在样本和标签定义。

这篇会把特征、样本、标签放在一个训练闭环里看，避免“模型升级但收益不稳”的情况。

## 这篇会覆盖什么

- 特征分层：用户、物品、上下文、交叉、序列
- 负采样、曝光偏差、标签时窗等核心问题
- 训练分布和线上分布不一致时怎么定位问题

## 当前提纲

- 特征体系搭建
- 样本与标签设计
- 训练到线上的一致性
