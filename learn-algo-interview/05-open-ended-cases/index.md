# 开放题、Case 题与业务追问

开放题考的不是标准答案，而是拆题方式。

你需要先定义目标，再明确约束、数据、方案、指标和风险，而不是马上给一个模型名。

## 这篇会覆盖什么

- 如何把开放业务题拆成算法问题
- 面对缺信息题目时如何主动澄清假设
- 如何回答“如果指标下降了怎么办”这类追问
