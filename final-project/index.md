# Final Project

这里会放 `zero2algo` 的最终实战项目。

目标不是再写一篇泛泛的“项目介绍”，而是把一个真实算法问题从业务目标、数据设计、指标体系、模型方案、系统约束到复盘闭环完整串起来。

## 当前规划

- 选一个业务目标明确、指标可量化的算法问题
- 同时覆盖离线实验、在线约束和收益归因
- 作为简历项目、面试项目和课程收束案例

## 候选方向

- 搜广推排序优化
- 广告投放预算与出价策略
- 内容风控策略迭代
- 供需匹配或调度优化
